Platform Setup Picker

Choose the simplest capture path before opening OBS.

Use The Normal Browser Extension When

  • The platform works in your normal browser.
  • You already have the right account signed in.
  • The supported-sites page says to open the site or pop out chat.

Start here for YouTube, Twitch, Facebook, Discord, Slack, WhatsApp, and most browser-based sources.

Use A Popout Or Chat-Only Window When

  • The platform has a separate popout chat URL.
  • The normal stream page is heavy or unreliable.
  • You need the chat window visible while streaming.

Check exact platform notes on the Supported Sites page.

Use The Desktop App When

  • Browser tabs are being hidden, throttled, or discarded.
  • You want source windows managed inside the app.
  • You are using an app-specific integration, such as some TikTok modes.

Desktop setup: Standalone App Guide.

Use A Source Page When

  • The platform needs a dedicated Social Stream source URL.
  • The setup asks for a channel, room, token, or OAuth flow.
  • You are using a WebSocket/API-style source instead of DOM capture.

Source pages can behave differently than normal browser capture, especially for send-back and events.
