Browser Extension vs Standalone App

Both work great — pick what fits your workflow.

Browser Extension

  • Uses your existing browser sign-ins automatically.
  • Fewer site-blocking hassles (runs within the site you’re viewing).
  • Fast to install via Web Store or manual install.
Keep in mind: For continuous capture, keep the site’s tab open and visible — many browsers throttle minimized/background tabs.

Standalone App

  • Captures in the background — no browser windows need to stay open or visible.
  • No extension required; auto‑updates; extra tools like always‑on‑top and transparency.
  • Offers additional direct WebSocket connection options.
Keep in mind: Some sites may try to limit app-based access and can require workarounds; you’ll sign in inside the app (separate from your browser session).

Manual Installation Guide

1

Download and Extract

Download the ZIP file and extract it to a folder on your computer.

Download ZIP
2

Open Browser Extensions Page

Navigate to your browser's extensions page:

chrome://extensions/ for Chrome
edge://extensions/ for Edge
brave://extensions/ for Brave
3

Enable Developer Mode

Toggle on "Developer mode" in the top-right corner of the extensions page.

4

Load Unpacked Extension

Click "Load unpacked" and select the folder where you extracted the files.

5

Start Using Social Stream Ninja

You're ready to go! Open a supported chat platform and click the Social Stream Ninja extension icon to enable it.

Firefox Installation (Limited Support)

1

Download and Extract

Download the ZIP file and extract it to a folder on your computer.

Download ZIP
2

Open Firefox Debugging Page

Navigate to Firefox's debugging page:

about:debugging#/runtime/this-firefox
3

Load Temporary Add-on

Click "Load Temporary Add-on" and select any file inside the extracted Social Stream Ninja folder.

4

Important Limitations

Note that this is a temporary installation. Your settings will not persist between browser sessions, and you'll need to reinstall each time you restart Firefox.

Linux AppImage Installation

Tips for Linux Users

Some distributions may encounter sandbox errors when running our AppImage. View our sandbox troubleshooting guide if you see errors about SUID sandbox configuration.

1

Download the AppImage

Download the AppImage file to your computer.

2

Make it Executable

Right-click the file and select Properties, then Permissions, and check "Allow executing file as program". Alternatively, use terminal: chmod +x SocialStreamNinja.AppImage

3

Run the Application

Double-click the AppImage to run Social Stream Ninja, or run it from terminal: ./SocialStreamNinja.AppImage

Frequently Asked Questions

The Chrome Web Store version is easier to install but only gets updated every few weeks due to Google's review process. The manual installation requires developer mode but gives you immediate access to the latest updates and features as soon as they're released.
If you're updating the extension by replacing files and reloading, your settings will be preserved. However, if you uninstall the extension completely, you will lose your settings. It's recommended to export your settings before a complete uninstall.
To update, download the latest version, replace the old files with the new ones, and then reload the extension or restart your browser. You do not need to uninstall the extension first. Backing up your settings first is suggested before updating.
This warning about Manifest v2 appears because Google has deprecated support for this version. We recommend updating to Manifest v3. Before updating, export your settings first, then install the v3 version of the extension. Both versions are currently available, but v2 has limited support and will eventually stop working.
The standalone app doesn't require a browser extension, automatically updates itself, and offers additional features like window pinning (always-on-top) and background transparency. It's also less likely to be affected by browser throttling issues when minimized.
