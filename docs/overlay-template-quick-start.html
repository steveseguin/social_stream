Template Quick Start

Pick an overlay template, attach your session ID, and add it to OBS as a browser source.

1. Confirm Chat First

Before setting up a template, make sure the dock receives messages for your session ID.

https://socialstream.ninja/dock.html?session=YOUR_SESSION_ID

2. Browse Templates

Open the template gallery and choose a layout that matches the kind of chat overlay you want.

3. Add Your Session ID

Use the same session ID that your chat source and dock use.

https://socialstream.ninja/templates/example.html?session=YOUR_SESSION_ID

4. Add The Browser Source

  1. In OBS, add a Browser Source.
  2. Paste the template URL.
  3. Set the browser source size to match the template or your scene.
  4. Refresh the source after changing URL options.

5. Adjust The Look

Most templates support URL options for layout, fonts, colors, message limits, and animation behavior.

More options are covered in the Templates Gallery and overlay customization guide.
