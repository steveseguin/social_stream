Getting Started: Adding Your First Chat Source

  1. Look at the left sidebar of the main application window.
  2. Click the button corresponding to the platform you want to add (e.g., YouTube Username, Twitch Username, etc.).
  3. A prompt will appear. Enter your username, Video ID, or URL as requested.
    • For most platforms, this is your username (e.g., YourChannelName).
    • For YouTube, you can also add by Video ID if you know it.
    • For unsupported sites, use Other chat sites and paste the direct chat URL.
  4. The source will be added to the main area. Now click the Activate source button (or Select Live Sources / Auto-find & Activate for YouTube channel sources).
  5. For some sites (like Facebook, TikTok, Instagram), you may need to use the Sign in button first or navigate within the popped-out window to find the live chat.
  6. Your chat messages will now start appearing in the Streaming Chat page! You can access this via the link at the top right of the main app window, labeled Streaming chat (dock & overlay). Copy this link into OBS as a Browser Source or Dock.

Key Features

  • Supports live automated two-way chat messaging with Facebook, YouTube, Twitch, Kick, Zoom, TikTok, Rumble, DLive, and dozens more.
  • Includes a "Featured Chat" overlay, with messages selectable via the dockable dashboard (auto or manual selection).
  • Supports bot commands and automated chat responses (custom logic via scriptable plugins).
  • LLM support (including native Ollama API) for AI moderation, chat assistance, RAG, and custom instructions.
  • Text-to-Speech (TTS) support for free, premium, and ultra-premium services.
  • Multi-channel source-icon support to easily differentiate streams.
  • Automatic message relaying between platforms.
  • No user login, API key, or permissions needed for chat capture from most sites.
  • Message queuing for later highlighting.
  • Secure data transport using VDO.Ninja's API - free and low latency!
