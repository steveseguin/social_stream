Request a Feature

Need support for a new platform or feature? Let us know!

While Social Stream Ninja already supports 100+ platforms, we're always open to adding more based on community demand. To request a new platform integration or feature:

  1. Check if it's already supported in our Supported Platforms list
  2. Join our Discord community and post in the #feature-requests channel
  3. Or create a GitHub issue with details about the request

Note: Not all requested platforms can be supported. We prioritize platforms with significant user bases and those that are technically feasible to integrate.

Common Support Questions

  • Disable browser throttling: Go to chrome://flags/#enable-throttle-display-none-and-visibility-hidden-cross-origin-iframes and set to "Disabled"
  • Also disable chrome://flags/#calculate-native-win-occlusion
  • Try the standalone desktop app version instead of the browser extension

If you see a blue debugging bar or auto-responses aren't working:

  • Make sure the pop-out chat window is open and visible
  • Try a Chromium-based browser (Chrome, Edge, Brave) instead of Firefox
  • Make sure you have permission to chat in the platform (some require subscription)
  • Start Chrome with --silent-debugger-extension-api flag to hide the blue bar

For security reasons, Discord, Slack, Telegram, WhatsApp, and Google Meet require you to enable their toggles in the extension menu settings.

Make sure to toggle on support for each platform you want to use, and check out our Advanced Features Guide for more details.
