Features

Discover all the powerful capabilities that make Social Stream Ninja the ultimate tool for managing your cross-platform social engagement.

Multi-Platform Chat Integration

Connect with viewers across all your streaming platforms simultaneously. Social Stream Ninja consolidates chat messages from dozens of platforms into a single, easy-to-manage dashboard.

  • Live two-way chat messaging with Facebook, YouTube, Twitch, Zoom, and more
  • No login or API keys needed for most platforms
  • Multi-channel source icons to differentiate between platforms
  • Automatic message relaying between platforms

Featured Chat Overlay

Highlight the best messages from your audience directly on your stream with beautiful, customizable overlays.

  • Select messages via the dockable dashboard
  • Auto or manual message selection
  • Customizable with CSS for perfect integration with your stream's aesthetic
  • Animation options including fade and swipe effects
  • Message queuing for organized highlighting

Bot Commands & Auto-Response

Create automated interactions with your audience through custom commands and responses.

  • Built-in commands for common interactions
  • Custom command logic with scriptable plugin file
  • Automated greetings and responses
  • Message filtering and moderation

AI & LLM Integration

Leverage the power of AI to enhance your stream interactions with intelligent responses and moderation.

  • Native Ollama API support
  • AI-based chat moderation
  • Smart responses using AI
  • RAG (Retrieval Augmented Generation) capabilities
  • Custom AI instructions for personalized interactions

Text-to-Speech

Bring chat messages to life with text-to-speech capabilities, making your stream more engaging and accessible.

  • Free system TTS support
  • Premium TTS services integration (Google Cloud, ElevenLabs, Speechify)
  • Multi-language support
  • Customizable voice, pitch, and speaking rate
  • OBS audio capture support

Deep Customization

Make Social Stream Ninja truly yours with extensive customization options for both functionality and appearance.

  • CSS styling for complete visual control
  • Pre-styled templates and themes
  • Custom JavaScript support
  • URL parameters for easy configuration
  • Source filtering and branding options

Powerful API

Extend Social Stream Ninja's capabilities with robust API integration options.

  • Webhook support for third-party integrations
  • Server API for remote control
  • Donation integration (Stripe, Ko-Fi, Buy Me A Coffee)
  • MIDI/Streamdeck control
  • OBS remote scene support
