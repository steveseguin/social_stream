Rumble API Alerts

Use your creator Live Stream API URL to relay Rumble chat, rants, followers, subscribers, gifted subs, and viewer counts without scraping the live page.

Connect a Rumble API URL

Paste the private URL from rumble.com/account/livestream-api. This source is read-only and only forwards normalized events into Social Stream.

This URL already includes your live stream key. No separate login is required. Sending messages is not supported through the API.
Keep this URL private. Anyone with it can read your live stream API data. Reset it from the Rumble dashboard if it has been shared accidentally.
Quick setup
  1. Open the Rumble API dashboard.
  2. Generate the Live Stream API URL for the user or channel you want to monitor.
  3. Paste it here and click Connect.
  4. Go live on Rumble if you want chat and live viewer data to appear.
