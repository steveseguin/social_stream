Rumble API Alerts for Social Stream

Paste your creator Live Stream API URL to relay Rumble chat, rants, followers, subscribers, gifted subs, and viewer counts without scraping the live page DOM.

  1. Open the Rumble API Dashboard (or click "Open API Dashboard" above)
  2. Generate your Live Stream API URL and copy it
  3. Paste it into the field below
  4. Click Connect
