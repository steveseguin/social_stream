What gets converted
Best fit: StreamElements or Streamlabs chat widgets with HTML, CSS, JS, fields, data, and local assets.
SSN namedata.displayName, data.nick
SSN messagedata.text, renderedText
SSN badgesdata.badges where URLs or badge labels exist
SSN platformservice
SSN message IDdata.msgId for delete-message support
SSN member/mod/VIP hintsSubscriber, moderator, VIP, or broadcaster style flags when detectable
SSN donations/mediadata.amount and attachment image data where available
Known limits
  • Chat widgets are the main target; goals, alerts, loyalty, store, tips, and account API features are only shimmed if their code falls back cleanly.
  • Widgets that require private StreamElements or Streamlabs API tokens, overlay-store state, full jQuery plugins, or platform emote APIs may need manual edits.
  • Local packaged images, fonts, audio, and video are embedded; remote URLs stay remote and may still fail if the source blocks OBS/browser access.
  • The export is a single HTML file with the session embedded, but ?session=... and &password=... in the OBS URL still override it.
